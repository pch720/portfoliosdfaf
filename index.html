#서울시_공중화장실_위치정보_제공 #급한불

프로젝트 설명

"급한불"은 서울시 공중화장실 위치데이터를 사용하여 지도에 나타내어 주는 사이트입니다.
또한, 공중화장실에 대한 커뮤니티가 존재하며, 작성된 게시글은 오피니언 마이닝을 통해 긍정과 부정반응으로 분류되어 나타내어 집니다.

적용기술

- 프레임워크 : Spring Framework
- 데이터베이스 : Oracle 11g
- 적용 공공데이터(OpenAPI) : 서울시 공중화장실 위치정보
- 기타 : Opinion Mining, Kakaomap API, Geolocation API
